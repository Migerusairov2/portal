# Portal

A modern developer portfolio and social platform built with **Django**, featuring user authentication, profile management, REST APIs, relational database design, and external data integration through web scraping.

---

## ✨ Features

* 🔐 User Authentication

  * Login, Register, Logout
  * Google OAuth Authentication
  * Github Authentication
  * Password Reset

* 👤 User Profiles

  * Editable developer profile
  * Profile image
  * Skills & Technologies
  * Social media links

* 📂 Project Showcase

  * Create and manage projects
  * Upload project images
  * Pin featured projects
  * Technology tags

* 📊 GitHub Integration

  * Synchronize public repositories
  * Repository statistics
  * Commit history
  * Automatic repository updates

* 💬 Social Features

  * Create posts
  * Like posts
  * Comment system
  * User avatars
  * Feed-style interface

* 🌐 REST API

  * Django REST Framework
  * JSON responses
  * API endpoints for application data

* 🗄 Database

  * Relational Database Management System (RDBMS)
  * Normalized database structure
  * Foreign key relationships
  * Efficient querying using Django ORM

* 🕸 Web Scraping Integration

  * Collect data from external sources
  * Process and display scraped content
  * Automated data retrieval
